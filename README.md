# Анализ паролей

Веб-приложение для оценки надёжности паролей и генерации безопасных комбинаций.

![Preview](preview.png)

## Концепция

Проект решает две задачи:

1. **Анализ пароля** — показывает, сколько времени потребуется на подбор вашего пароля методом грубой силы (при скорости 100 млрд попыток в секунду). Оценка учитывает:
   - Длину пароля
   - Размер алфавита (набор используемых символов)
   - Наличие словарных слов и паттернов (через zxcvbn)

2. **Генератор паролей** — создаёт случайные пароли с настраиваемыми параметрами:
   - Длина (от 8 до 64 символов)
   - Набор символов: строчные (a-z), заглавные (A-Z), цифры (0-9), спецсимволы (!@#$)

## Техническая реализация

- **Стек:** Чистый HTML, CSS, JavaScript (без фреймворков)
- **zxcvbn** — библиотека от Dropbox для оценки надёжности паролей (подключается через CDN)
- **Алгоритм взлома:** Оценивается время перебора всех комбинаций: `pool_size^length / 100_000_000_000` попыток/сек
- **Генерация:** Криптографически случайные символы с обязательным включением выбранных типов

## Запуск

Просто откройте `index.html` в браузере. Никаких зависимостей и сервера не требуется.
